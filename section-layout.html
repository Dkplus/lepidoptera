3.1 Navbar

Horizontal navigation bar from MaterializeCSS.

Please have a look at the documentation of MaterializeCSS for many awesome build in features.

Modifiers should be added to the nav element if it is not wrapped by a .navbar-fixed. Otherwise it should be added to the .navbar-fixed wrapper.

To initialize the plugin you need to include js/waterfall-nav.js and call $('.nav-waterfall-2x').waterfallNav();. You may pass {enlargeOnScrollUp: true} to change the behavior on scrolling up.

3.1.1 Badge

Support for badges within the Navbar is built in in MaterializeCSS. But its missing support for badges near to Materialize Icons (see the Default styling example). This has been added. Together with class circle it can be used to create a nice looking notification counter.

Examples
Default styling
  • Missed phones 2
  • mail_outline 1
  • 2
.badge--raised

Moves the badge to the top right side of the element. This is currently supported only with numeric content.

  • mail_outline 1
  • 2

       </a>
     </li>
     <li>

         <i class="fa fa-envelope-o"></i>

 </div>
</nav>

3.2 SideNav

Slide out menu from MaterializeCSS. It has been extended with some modifiers that allow a more flexible usage of the side-nav.

The two most noteworthy things are the possibility to always have a small side-nav that expands on hover (See .side-nav--mini and to add an logo to the sidebar.

For most use cases you can keep up using the normal markup from Materialize, but for some modifiers you need to put the side-nav class at a div and put the ul inside of this div:

<div class="side-nav">
 <ul>
   <li><a href="#!">Label</a></li>
 </ul>
</div>

Please have a look at the documentation of MaterializeCSS especially for the initialization with jQuery and the Dropdown HTML Structure.
